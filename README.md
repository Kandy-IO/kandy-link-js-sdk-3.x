# @kandy-io/link-sdk

## Install

### Using npm :

`$ npm install @kandy-io/link-sdk`

#### Install a particular build :

To install a particular build branch or beta branch with the Github URL, run:

`$ npm install https://github.com/Kandy-IO/kandy-link-js-sdk#<build number or "beta"> --save`

### In the browser :
```html
<script src="https://unpkg.com/@kandy-io/link-sdk@3.5.0/dist/kandy.js"
  integrity="sha384-q5JfNZcsKdP2Upr03yRyvCg9dSDhkWdYgaE5HkFskg+FZNxXSIJBbw39PaR2zXb/"
  crossorigin="anonymous">
</script>
```
