# @kandy-io/link-sdk-3.x

## Install

### Using npm :

`$ npm install @kandy-io/link-sdk-3.x`

#### Install a particular build :

To install a particular build branch or beta branch with the Github URL, run:

`$ npm install https://github.com/Kandy-IO/kandy-link-js-sdk-3.x#<build number or "beta"> --save`

### In the browser :
```html
<script src="https://unpkg.com/@kandy-io/link-sdk@3.11.0/dist/kandy.js"
  integrity="sha384-6d92g4IhwdLlv1qMsAzqXHSc2XlHZ6+I8Xf3ve3LCD9P03N3aJxYdq/brDUH7Atr"
  crossorigin="anonymous">
</script>
```
